Pen and Paper

An sample e-commerce website that uses Gatsby.js for the framework, TailwindCSS for styling, and Snipcart for e-commerce functionality. It also leverages the image processing features of Cloudinary. Hosted on Vercel.

Reading List

https://reading-list-92431.web.app

An interactive list of books that are on my “to-read” list made with React.js. Back-end functionality done using Firebase.

Travel Bucketlist

https://tims-travel-bucket-list.onrender.com

A visual list of my personal travel bucketlist. The page is hosted on Render. Content is powered by Hygraph — formerly GraphCMS — which primarily uses GraphQL to interact with the API. Styled with TailwindCSS. Page is hosted on Render and deployed through Github.

My Mom's Personal Blog

https://bridgetangonan.com

Created a website for my mother where she can document and share her thoughts. Front-end framework used for the project is Next.js. Styled using TailwindCSS. Content management system provided by Contentful and hosted on Netlify and deployed through GitHub.

My Wife's Recipe Site

https://ling-cookbook.netlify.app/

Built a website where my wife can add recipes through Prismic.io and refer to them conveniently on mobile while cooking. Items for ingredients and procedures are toggle-able to indicate completed status. Front-end framework used is Next.js. Site is hosted on Netlify and deployed through GitLab.
